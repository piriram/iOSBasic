# 네트워크 이해하기

### Concurrency(동시성)
- 여러개의 작업이 동시에 일어나 보이도록 하는 것
- 작업들이 순서에 상관없이 동시에 수행될 수 있음
- 밥먹으면서 공부하기
  - 실제로는 동시에 진행되지않음
  - 밥 한숟가락 -> 두줄 -> 한숟가락 -> 두줄

### Tread
- 각각의 작업들은 쓰레드에서 실행됨
- **⭐️메인 쓰레드⭐️ : 화면 표시 및 사용자 인터랙션을 받는 작업 수행**
  - 데이터를 화면에 보여주기
  - 버튼 클릭 이벤트 감지
- **백그라운드 : 메인 쓰레드 외에 작업들**
  - 네트워크에서 데이터를 다운받을 때나
  - 디비에서 데이터를 읽어 온다던가
- **custom : 지정된 스레드를 통해서 작업할 때 사용**
  - 카메라를 통해서 들어온 영상 데이터 가공시
  - 오디오 데이터 변조시
