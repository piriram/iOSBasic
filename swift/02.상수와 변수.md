# 02. 상수와 변수

## 1. 상수
* `키워드` let
* `상수의 선언` let 이름: 타입 = 값<br>
_:과 타입 사이에 꼭 한칸을 떼어주자_<br>
${\color{blue}값과 타입이 명확하다면 타입은 생략가능}$
* `차후에 변경이 불가능`<br><br>

## 2. 변수
* `키워드` var
* `변수의 선언` var 이름: 타입 = 값<br>
* `차후에 변경이 가능`
* `나중에 할당하려고 하는 상수나 변수는 타입을 꼭 명시
```
let sum: Int
var nickName: String
// 나중에 할당하기 위해 선언만 해놓음
```
